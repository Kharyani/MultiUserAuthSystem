# Multi-User Authentication System (Python)

## 🎯 Objective
A secure multi-user authentication system using Python with JSON-based storage. Users can register, login, reset passwords, and delete accounts.

---

## 🛠 Features
- User registration with unique username & email
- Strong password validation (8+ chars, uppercase, lowercase, number)
- Email format validation
- Password hashing using SHA-256 (secure storage)
- Login with username or email
- Password reset functionality
- Account deletion
- JSON-based persistent storage
- Command-line interface (CLI)

---

## ⚙️ How to Run

1. Make sure you have **Python 3.7+** installed
2. Clone or download this repository
3. Open terminal/command prompt in project folder
4. Run:

```bash
python auth_system.py
